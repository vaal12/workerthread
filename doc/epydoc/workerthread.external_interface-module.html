executeInWorkerThread(function2execute, delay_ms=0, result_callback=None)

source code 

This function posts a function2execute function to workerthread queue for execution.

Parameters:
  • function2execute (function pointer or lambda expression) - function to execute in the workerthread
  • delay_ms (delay in milliseconds) - delay after which function2execute will be executed.
  • result_callback (function pointer or lambda expression) - function to execute in the workerthread when result of function2execute is available. Function result_callback will receive a result returned by function2excute
Returns:
nothing Note: delay between call to this function and actual execution cannot be exactly delay_ms. It is only guaranteed that function2execute will be executed on or after delay_ms is passed. For more on the queueing/execution of functions in workerthread, see package documentation.

Example in - samples/example1.py

finishThreadAndWaitForCompletion()

source code 

This function notifies the workerthread that it should terminate and then waits for the workerthread to terminate. Note: This function will block until all delayed functions due to the moment of completion are executed (if there are functions due after stop point, they may be silently discarded) and all queued (non delayed) before this call functions are also executed (all non delayed functions after this call will be silently discarded). See any sample on this function.

dumpArgsDecorator(func)

source code 

This is a helper decorator to be used for debugging. This decorator dumps out the arguments passed to a function before calling it.

Code is taken from http://wiki.python.org/moin/PythonDecoratorLibrary Note: at the moment it only dumps arguments to stdout, in future I will add dumping those to log (via loggin module).

Example of usage: samples\example_decor1.py

trace(f)

source code 

Decorator, which will trace with code line numbers execution of the function. Very useful for debugging. May be chained with other decorators:

       @workerthread.executeInWorkerThreadDecorator    
       @workerthread.trace
       def traceExampleFromWorkerThread(message):
               print "MEssage"
               print message
               return None

See example of usage in samples\example_decor1.py

Code taken from http://wiki.python.org/moin/PythonDecoratorLibrary#Line_Tracing_Individual_Functions
